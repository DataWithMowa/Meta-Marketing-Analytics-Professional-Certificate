# Meta Marketing Analytics Professional Certificate

This repository documents my learning journey, laboratory exercises, practical assignments, exams, and hands-on industry projects completed during the **Meta Marketing Analytics Professional Certificate** program. 

Developed by marketing analytics experts at Meta and Aptly, this rigorous eight-course series focuses on leveraging data-driven strategies, statistical methodologies, and modern analytical tools to optimize marketing performance and drive business growth.

---

## 📈 Key Learning Objectives

* **Data Engineering & Analysis:** Apply the **OSEMN framework** (Obtain, Scrub, Explore, Model, iNterpret) to approach complex analytics questions.
* **Campaign Optimization:** Design robust experiments, conduct A/B testing, and interpret Bayesian statistics to evaluate advertising effectiveness.
* **Platform Mastery:** Utilize **Meta Ads Manager** to deploy campaigns, run split tests, and interpret performance metrics.
* **Data Storytelling:** Translate raw text and numeric datasets into meaningful business insights and executive-ready dashboards.

## 🗂️ Program Structure & Core Skills

### 📂 Course Modules & Topics Covered
1. **Foundational Marketing Principles:** Understanding business metrics, marketing mixes, and audience segmentation.
2. **Data Collection & Cleaning:** Identifying data sources, filtering anomalies, and preparing databases for structured analysis.
3. **Exploratory Data Analysis (EDA):** Using Python and Pandas to sort, explore, and analyze high-volume marketing datasets.
4. **Visual Analytics:** Constructing interactive, data-driven dashboards using Tableau to convey clear marketing narratives.
5. **Marketing Science & Experiments:** Executing A/B tests, evaluating the sales funnel, and analyzing descriptive and inferential statistics.

### 🧪 Applied Learning Projects Portfolio
This folder contains the practical applications of my coursework, including:
* Cleaning and processing campaign datasets using advanced spreadsheets.
* Building data pipelines with Python to evaluate marketing attribution models.
* Creating visual trend analyses in Tableau to track customer acquisition costs (CAC) and lifetime value (LTV).
* Evaluating real-world ad experiments to optimize budget allocation and return on ad spend (ROAS).
